# CONTROL-PANEL-Boiler-Temperature-Control-Panel-Using-Arduino-
A smart Arduino-based Boiler Temperature Control Panel designed to monitor and control boiler temperature efficiently. This project helps maintain safe temperature levels automatically and demonstrates basic industrial automation concepts using sensors and microcontrollers.
# Project Overview
This system continuously monitors boiler temperature using a temperature sensor and displays real-time values. When the temperature exceeds the predefined limit, the controller can trigger cooling/safety mechanisms automatically.
The project was developed using:
- Arduino UNO
- Wokwi Simulation
- Arduino Programming
# Features
Real-time temperature monitoring  
Automatic temperature control  
LCD display output  
Safe and efficient operation  
Arduino-based automation  
Easy simulation using Wokwi

#Working Principle
1. The temperature sensor continuously reads boiler temperature.
2. Arduino processes the sensor data.
3. Current temperature is displayed on the LCD.
4. If temperature exceeds the threshold:
   - Warning indication is activated
   - Control mechanism starts automatically
5. System maintains safe boiler operation.
